Description of test packages
============================================

Sample Packages downloaded from http://deffinum.com/deffinum-explained/technical-deffinum/golf-examples/

* singlescobasic.zip - Single SCO with basic runtime calls. DEFFINUM 1.2.
* singlesco_deffinum12.zip - Single SCO content packaging example. DEFFINUM 1.2.
* RuntimeMinimumCalls_DEFFINUM12.zip - Multi-SCO packaging example. DEFFINUM 1.2.
* RuntimeBasicCalls_DEFFINUM20043rdEdition.zip - Multi-SCO packaging example. DEFFINUM 2004 3rd edition.
* RuntimeMinimumCalls_DEFFINUM12-mini.zip - modified 2 SCO version of RuntimeMinimumCalls_DEFFINUM12.zip

These packages were downloaded from http://deffinum.com/ website,  the website
disclaimer states that *Content on this site is licensed under a Creative
Commons Attribution 3.0 License*. http://creativecommons.org/licenses/by/3.0/


Other test packages
* baddeffinum.zip - contains a fake imsmanifest.xml inside a directory, used for validation check.
* invalid.zip - zip file with an single html file, no DEFFINUM config files, used for validation check.
* validdeffinum.zip - non functional package with an imsmanifest.xml, used for validation check.
* validaicc.zip - non functional package with AICC config files, used for validation check.
* complexdeffinum.zip - copied from: https://github.com/jleyva/deffinum-debugger.
* singlescobasic_missingorg.zip - copy of deffinum.com package but with missing org definition.
